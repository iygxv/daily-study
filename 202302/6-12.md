- 闲暇之余
  题目:
  ```js
  var F = function () {}
  Object.prototype.a = function () {}
  Function.prototype.b = function () {}
  
  var f = new F()
  
  // 问实例f 有 a， b方法吗？
  
  // 只有a，没有b方法
  ```

  因为函数是一等公民， 其实也可以算作一个对象，因此

  ```js
  function a = {}
  a instanceof Function
  
  a instanceof Object
  ```

  返回题目解释

  ```js
  f.__proto__ === F.prototype
  而
  F => function () {}
  即
  F.__proto__ === Function.prototype
  所以
  f instanceof Function // false
  ```

  因此才没有a方法
